## 🖥️ About

**DvorakMSDOS** simulates the look and feel of an old-school MS-DOS environment,
right inside your Windows command prompt — no emulator needed.
It boots straight into a login screen, drops you at a retro main menu, and
lets you launch apps, play quiz games, and browse "the web" — all without
leaving `.bat` files.

---

## 🚀 Getting Started

1. **Clone or download** this repository.
2. **Double-click** `DvorakMSDOS.bat` (or run it from a command prompt).
3. The title bar will show you a helpful hint 😉
4. Log in and enjoy the ride.

> ⚠️ **Requires Windows** with a standard `cmd.exe` environment.

---

## 🔐 Login

When the simulator starts, you are greeted by a login screen.
Your credentials are:

| Field    | Value   |
|----------|---------|
| Username | `admin` |
| Password | `1234`  |

> 💡 **Tip:** The window title gives you a nudge in the right direction when
> you first launch the program.

## 📌 Main Menu

After logging in, you land on the **Main Menu**, which gives you access to
four built-in applications:

```
╔══════════════════════╗
║     MAIN  MENU       ║
╠══════════════════════╣
║  [1]  App 1          ║
║  [2]  App 2          ║
║  [3]  App 3          ║
║  [4]  App 4          ║
╚══════════════════════╝
```

Each app is a self-contained batch module — lightweight, fast, and very retro.

## 🏁 Start Menu

The **Start Menu** is your gateway to extra goodies:

  unexpected. 👀
